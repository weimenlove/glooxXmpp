# glooxXmpp
study xmpp based on gloox

## 获取gloox ##

http://camaya.net/gloox

## 使用gloox ##

1、直接将gloox的源代码放到自己的源代码中进行开发

2、将gloox使用对应的编译器进行编译，在自己的源代码中使用gloox的库和头文件
